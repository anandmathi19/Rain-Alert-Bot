# 🌧 Rain Alert Bot – Python

A Python automation project that checks weather forecasts and sends alerts when rain is expected. This helps users stay prepared by reminding them to carry an umbrella or plan their day accordingly.

---

## 🚀 Features

- Fetches weather forecast data using a weather API
- Detects upcoming rain conditions
- Sends alerts or notifications to the user
- Simple automation using Python

---

## 🛠 Technologies Used

- Python
- Weather API
- Requests Library
